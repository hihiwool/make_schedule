# Project Overview: GMPKK 스마트 ROSTER 뷰어

엑셀 형식의 ROSTER 데이터를 붙여넣으면 사용자를 선택하여 달력 형태의 근무표로 시각화해주고, 이미지로 저장할 수 있는 웹 애플리케이션입니다.

## 주요 기능
- **데이터 파싱:** "OO년 OO월" 정보가 포함된 엑셀 텍스트 데이터를 분석하여 인원별 근무 데이터를 추출합니다.
- **달력 시각화:** 선택한 사용자의 한 달 근무를 요일에 맞춰 달력 형태로 보여줍니다.
- **근무 타입별 색상:** A(오전), P(오후), S(특근), O(휴무) 등 근무 코드에 따른 자동 색상 코딩을 지원합니다.
- **이미지 저장:** `html2canvas`를 활용하여 생성된 근무표를 고화질 PNG 이미지로 저장할 수 있습니다.
- **반응형 디자인:** 모바일 및 데스크탑 환경 최적화.

## 기술적 세부 사항
- **Frontend:** Vanilla JS, HTML5, CSS3.
- **Libraries:** html2canvas (이미지 캡처).
- **Architecture:** 단일 HTML 파일 구조로 빠른 로딩 및 배포 최적화.
