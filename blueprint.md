# GMPKK 스마트 ROSTER 뷰어 프로젝트

## 프로젝트 개요
엑셀 형태의 ROSTER 데이터를 분석하여 사용자별 맞춤형 달력 뷰를 제공하고, 이미지로 저장할 수 있는 웹 어플리케이션입니다.

## 현재 구현된 기능
- 엑셀 데이터 파싱 (년/월 자동 인식)
- 사용자별 근무 달력 생성
- 근무 타입별 색상 구분 (A, P, S, O, ETC)
- html2canvas를 이용한 달력 이미지 저장

## 추가 예정 기능 (현재 작업 중)
- **동료 근무자 표시 기능**: 각 날짜별로 같은 시간대(접두어 일치)에 근무하는 동료들을 달력에 표시.
    - `A` 계열(A1, A2 등)은 함께 표시.
    - `P` 계열(P1, P2 등)은 함께 표시.
    - `O`, `Y`는 휴무로 간주하여 동료 표시 제외.

## 작업 계획
1.  **CSS 수정**: 동료 이름을 표시할 `.colleagues` 클래스 스타일 추가.
2.  **JS 로직 수정**: `render()` 함수 내에서 동일한 날짜, 동일한 근무 타입 접두어를 가진 다른 사용자들을 검색하는 로직 추가.
3.  **데이터 처리 보완**: `Y` 타입을 휴무(`O-type`)로 처리하도록 로직 보완.
4.  **검증**: 여러 명의 데이터가 입력되었을 때 동료들이 올바르게 표시되는지 확인.
